Build a full-stack web app called “FreshMood”.

FreshMood is a modern restaurant QR ordering and order management system for dine-in customers.

The concept:
When customers dine in at FreshMood, every table has a unique QR code. When customers scan the QR code using their phone camera, it opens a mobile-first web application displaying the FreshMood menu.

The app automatically detects and stores the table number from the QR code URL/session.

Customers can:
- Browse menu categories
- View product images
- See prices
- View “Best Seller” labels
- Add items to cart
- Customize quantity and notes
- Checkout directly from their phone
- Pay using multiple payment methods

Restaurant staff can:
- Receive incoming orders in real time
- Manage kitchen workflow
- Update order statuses
- Manage menu items
- View analytics and sales reports

CUSTOMER FLOW
==================================================

1. Customer scans QR code
2. Opens FreshMood web app
3. Table number automatically detected
4. Customer browses menu
5. Customer adds items to cart
6. Customer reviews order summary
7. Customer selects payment method
8. Customer completes payment
9. Order sent to kitchen dashboard in real time
10. Kitchen prepares order
11. Staff updates order status
12. Customer sees live order tracking
